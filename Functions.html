What is a function in c++?

A function is a block of code which only runs when it is called.


Functions are used to perform certain actions, and they are important for reusing code: Define the code once, and use it many times.


Creating a function


C++ provides some pre-defined functions, such as main(), which is used to execute code. But you can also create your own functions to perform certain actions.


To create (often referred to as declare) a function, specify the name of the function, followed by parentheses () a then ollowed by curly brackets {};


void Main () {


// some code


}

There are two types of main data_types in functions one being void and the other being int but you are also able to use other data_types but these are not relavant.

Void - The void data_type means that when the function finishes the data will not be returned or stored outside of the function.

Int - The int data_type means that the data used in the function has a data return and also using int means that you are able to acceess the data from a different function.


All of the code written iside of the curly brackets will be run once the function has been called, also the data stored inside of the function can be trasfered between functions but only if the data_type is not void as void means that there is no return value.
